## 🚀 Deploy to GitHub Pages

1. Upload all files to your GitHub repository root
2. Go to **Settings → Pages → Source → Deploy from branch → main / root**
3. Site will be live at `https://shalimoosavi.github.io/SAYANJALI_NEXUS/`
